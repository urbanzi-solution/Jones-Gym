# File stuctures

```
/project-root
│
├── /node_modules          # Contains all npm packages
│
├── /public                # Publicly accessible files
│
├── /src                   # Source files
│   ├── /app  
|   |    ├── (pages)       # Application pages
│   │           ├── /gallery       # Gallery related components
│   │           ├── /member-add    # Member addition components
│   │           ├── /member-profile # Member profile components
│   │           ├── /notification   # Notification components
│   │           ├── /reports       # Reporting components
│   │           ├── /staff         # Staff management components
│   │           ├── /translations   # Translation components
│   │           ├── /styles        # Styles for the application
│   │           ├── layout.js      # Main layout component
│   │           └── page.jsx       # Main page component
│   │
│   ├── /components        # Reusable components
│   │
│   └── /styles            # Global styles
│
├── .gitignore             # Git ignore file
├── database_setup.md      # Database setup instructions
├── jsonconfig.json        # JSON configuration file
├── next.config.mjs        # Next.js configuration file
├── package.json           # Project metadata and dependencies
├── package-lock.json      # Exact versions of npm dependencies
├── postcss.config.mjs     # PostCSS configuration file
├── README.md              # Project documentation
└── server.md              # Server setup instructions

```
